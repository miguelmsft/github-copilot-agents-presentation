Local Agents (VS Code)

Run interactively inside VS Code on your machine.

  • Full access to your workspace, extensions, and MCP servers
  • Real-time inline diffs as the agent edits
  • Richest tool access of any agent type
  • Stops when you close VS Code

Best for: Brainstorming, iterating, debugging — tasks where you want to stay in the loop.

Local agents are the most interactive option. They run right in your editor, so you see everything happening in real time. You get inline diffs for every change, full access to VS Code extensions, and you can steer the agent at any point. The tradeoff is that they stop when you close VS Code, and they're limited to your local machine.
