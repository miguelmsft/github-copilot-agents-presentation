agent: 'agent'
description: 'Generate unit tests for a function'
tools: ['search/codebase']
---

Analyze the selected function and generate focused unit tests.

## Strategy
1. Core functionality tests
2. Input validation and boundary tests
3. Error handling tests

Target function: ${input:function_name:Which function?}
Framework: ${input:framework:jest/pytest/etc}

Invoke with: /generate-tests in chat

Prompt files are like saved templates for common tasks. Unlike custom instructions that apply automatically, prompt files are invoked manually — you type a slash command to use one. They support input variables with the ${input:name:placeholder} syntax, making them reusable with different parameters. This example generates unit tests — you invoke it, provide the function name and testing framework, and the agent follows the template.
