Cloud Agent — Three Modes

As of April 2026, the cloud agent supports three distinct modes:

  1. 🔍 Research Mode — Investigate codebases, answer deep questions, no code changes
  2. 📋 Planning Mode — Create implementation plans, review before coding
  3. 💻 Implementation Mode — Make code changes, iterate, then create PR
"Copilot cloud agent is no longer limited to pull-request workflows."
The cloud agent used to be just about creating PRs. Now it can do research and planning too. Research mode lets you ask complex questions about a repository without making any changes. Planning mode generates a detailed plan you can review and iterate on before any code is written. Implementation mode is the classic workflow — code changes, testing, and PR creation. This makes the cloud agent much more versatile.
