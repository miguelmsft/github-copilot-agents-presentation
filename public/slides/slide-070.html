Agent Handoffs — Multi-Step Workflows

Custom agents can hand off to each other, creating guided workflows:

📋 PLANNER
Read-only
→
handoff
⚙️ IMPLEMENTER
Full tools
→
handoff
🔍 REVIEWER
Read-only

Each agent has its own instructions, tools, and model preference.

After a response completes, handoff buttons appear for the next step.

Handoffs let you chain agents together. A planner agent creates the implementation plan using read-only tools. When done, a "Start Implementation" button appears. Clicking it transitions to an implementer agent with full tool access. After implementation, a "Review" button transitions to a reviewer agent. Each agent has its own persona, tools, and model. This is a powerful pattern for structured workflows — plan, implement, review.
