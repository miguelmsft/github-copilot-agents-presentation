🏁 sessionEnd HOOK

Here's how hooks fit into the agent lifecycle. Session start and end hooks bookend the session — great for setup and cleanup. The user prompt hook fires when you type a message. The real power is in preToolUse and postToolUse — they fire before and after every tool the agent uses. The preToolUse hook is special because it can actually block the tool from executing. IMPORTANT caveat: on Cloud Agent and CLI, only the DENY decision is currently processed. ALLOW and ASK may work differently or not at all on those surfaces — VS Code has the broadest support for all three.
