MCP Apps — Interactive UI

MCP Apps allow tool calls to return interactive UI components directly in the chat.

Instead of: "Here are 200 rows of data as text..."

You get: An interactive data table with sorting, filtering, and clicking.

Examples: Dashboards, flame graphs, feature flag selectors, 3D scenes, component previews.

MCP Apps are a recent extension to MCP that bridges the gap between text responses and rich interfaces. Instead of the AI summarizing data in text, it can return interactive widgets — sortable tables, clickable charts, drag-and-drop lists. These render in sandboxed iframes right in the chat. It's supported in VS Code, Claude, and ChatGPT. This makes MCP servers much more powerful for data visualization and interactive workflows.
