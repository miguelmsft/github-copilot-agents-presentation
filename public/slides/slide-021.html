Permission Levels

LevelWhat It Does
DefaultAgent asks before running tools that modify files
BypassAuto-approves all tool calls (no dialogs)
Autopilot (Preview)Auto-approves AND auto-responds to questions
⚠️ Start with Default until you trust the agent with your codebase.
Permission levels control how much autonomy you give the agent. Default mode means you approve every potentially destructive action — like running a shell command or deleting a file. Bypass mode auto-approves everything, which is faster but riskier. Autopilot goes even further — it auto-responds to the agent's questions, so it never stalls. For beginners, stick with Default until you're comfortable with what the agent does.
