Agent Mode: How Billing Works

Each prompt you enter counts as one premium request.

The agent's follow-up actions — tool calls, file edits, iterations — do NOT count.

You're billed for asking, not for the work Copilot does.
Important practical detail. In agent mode, only your prompts cost premium requests. If the agent takes 50 actions to complete your task — reading files, editing code, running tests, fixing errors — those follow-up actions are free. Only the prompts you explicitly type count toward your quota. This makes agents very cost-efficient for complex tasks.
