Agentic Workflows (GitHub Actions)

AI-powered automation in GitHub Actions, written in Markdown:

---
name: "Daily Issues Report"
on:
  schedule: daily on weekdays
engine: copilot
safe-outputs:
  create-issue:
    title-prefix: "[daily-report] "
---

Create a daily summary of open issues for the team.
Include new issues, closed issues, and stale issues.

Compiles to GitHub Actions YAML. Currently in technical preview.

Agentic workflows are a separate but related system that brings AI agents into GitHub Actions. You write a Markdown file with natural language instructions, and it compiles to a GitHub Actions YAML workflow. The agent runs on a schedule or in response to events — like a daily issues report, automated PR review, or issue triage. It has a defense-in-depth security model with read-only tokens, network firewalls, and safe output guardrails. This is currently in technical preview.
