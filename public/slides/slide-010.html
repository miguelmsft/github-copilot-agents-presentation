Here's the loop visualized. Your prompt kicks it off. The agent reads files to understand what needs to change, then acts by editing code or running commands, then validates by checking for errors or running tests. If it finds problems, it loops back and tries again. This cycle can repeat many times — the agent is persistent. It doesn't give up after one try.
