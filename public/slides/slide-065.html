Here's the full picture. Seven layers, each serving a different purpose. Custom instructions are always on — your coding standards. Prompt files and custom agents are on-demand — invoked when you need them. Skills are context-aware — loaded automatically when relevant. Hooks are lifecycle-based — deterministic code at specific points. MCP provides external tool connections. And agentic workflows bring AI into GitHub Actions for CI/CD automation.
