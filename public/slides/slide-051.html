Skills with Scripts

Skills can bundle executable scripts:

.github/skills/image-convert/
├── SKILL.md
└── convert-svg-to-png.sh 
---
name: image-convert
description: Converts SVG images to PNG format.
allowed-tools: shell
---

Run the `convert-svg-to-png.sh` script from this
skill's base directory, passing the SVG file path
as the first argument.

⚠️ Only use allowed-tools: shell for trusted skills!

Skills aren't limited to text instructions — they can include scripts that the agent runs. This image conversion skill bundles a shell script and tells the agent to run it. The allowed-tools: shell field pre-approves terminal execution, so the agent won't ask for confirmation. This is powerful but dangerous — only use it for skills you've thoroughly reviewed.
