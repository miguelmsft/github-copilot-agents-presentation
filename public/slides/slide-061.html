Example: Audit Logging

Log every tool use for compliance:

{
  "version": 1,
  "hooks": {
    "sessionStart": [
      { "type": "command", "bash": "./audit/log-session.sh" }
    ],
    "userPromptSubmitted": [
      { "type": "command", "bash": "./audit/log-prompt.sh" }
    ],
    "preToolUse": [
      { "type": "command", "bash": "./audit/log-tool.sh" }
    ],
    "sessionEnd": [
      { "type": "command", "bash": "./audit/log-end.sh" }
    ]
  }
}

Creates a complete audit trail of every agent interaction.

For compliance-regulated environments, hooks let you create complete audit trails. This config logs every session start, every prompt, every tool use, and every session end. Your scripts can write to log files, send to a SIEM system, or push to a monitoring service. GitHub recommends redacting sensitive data — the tutorial shows regex patterns for removing tokens and API keys from logs.
