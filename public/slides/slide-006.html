What Is GitHub Copilot?

GitHub Copilot is an AI coding assistant that helps you write code faster.

It started as inline code completions — suggestions as you type.

Then it added chat — ask questions, get answers.

Now it has agents — autonomous task execution.

Quick refresher for anyone new. Copilot launched as a code completion tool — you type, it suggests. Then came Copilot Chat, where you could have a conversation about code. Agents are the third evolution — and the most transformative. Instead of just suggesting, agents *do the work*.
