Cloud Agent — Built-in Security

Every cloud agent session automatically runs:

  • ✅ CodeQL — Identifies security vulnerabilities in generated code
  • ✅ Secret Scanning — Detects API keys, tokens, and passwords
  • ✅ Dependency Checking — Checks new deps against GitHub Advisory Database
  • ✅ Copilot Code Review — Self-reviews its own changes

No additional license required. Can be disabled by repo admins.

This is a major advantage of the cloud agent over local agents. Security scanning is built in and runs automatically. Before the agent even opens a PR, it checks for hardcoded secrets, vulnerable dependencies, and code security issues. It even reviews its own code with Copilot code review. This doesn't replace human review — the PR is still a draft — but it catches many issues before a human ever sees them.
