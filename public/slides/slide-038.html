Three Core Primitives

🔧
Tools
Executable functions the AI can call

Controlled by the model

Example: Search flights
📄
Resources
Read-only data sources for context

Controlled by the app

Example: File contents
💬
Prompts
Pre-built instruction templates

Controlled by the user

Example: "Plan a trip"
MCP servers expose three types of capabilities. Tools are the most important — they're actions the AI can take, like searching flights or querying a database. The model decides when to call them. Resources are read-only data — like file contents or database schemas — that provide context. The application controls these. Prompts are reusable templates that the user selects. In practice, tools are what you'll use most.
