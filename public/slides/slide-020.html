Local Agent Features

  • Inline diffs — See changes as colored highlights in your editor
  • Checkpoints — Automatic snapshots let you roll back
  • Chat participants — @vscode, @terminal for specialized help
  • Image support — Attach screenshots and UI mockups to prompts
  • Subagents — Delegate subtasks to isolated agent instances
  • BYOK models — Bring your own API keys for any supported model
VS Code local agents have the richest feature set. Inline diffs let you see exactly what changed. Checkpoints create automatic snapshots so you can undo. You can attach images — paste a screenshot of a bug and say "fix this." And subagents let the main agent delegate complex subtasks to isolated workers that don't pollute the main conversation.
