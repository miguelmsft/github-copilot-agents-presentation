Copilot CLI — What It Is

A standalone command-line agent that runs directly in your terminal.

  • Separate from VS Code — no GUI needed
  • Can run in the background while you keep working
  • Supports multiple parallel sessions
  • Continues running even when VS Code is closed
The Copilot CLI is a terminal-native agent. You launch it with the `copilot` command and interact via text. The killer feature is background execution — you can kick off a task and keep working on something else while the agent handles it. You can even run multiple CLI sessions in parallel on different tasks. Think of it as having multiple junior developers working in the background.
