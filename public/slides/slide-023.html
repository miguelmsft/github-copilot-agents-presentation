Copilot CLI — Interactive Mode

Start a conversation in your terminal:

copilot
> Add error handling to all API endpoints
  • Have multi-turn conversations with follow-up prompts
  • Press Shift+Tab to toggle between Agent and Plan mode
  • Steer the agent mid-task by typing additional messages
The first way to use the CLI is interactive mode. Just type `copilot` and start talking. It works like a chat — you give it a task, it works on it, and you can steer with follow-up messages. You can toggle into Plan mode with Shift+Tab if you want a structured plan before coding. This is great for exploratory work where you want to stay in the loop.
