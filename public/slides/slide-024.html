Copilot CLI — Programmatic Mode & Key Commands

Single prompt (for scripting / CI):

copilot -p "Fix all failing tests" --allow-all-tools

Key session commands:

copilot --continue     # Resume last session
copilot --resume       # Pick a session to resume
/model                 # Change AI model
/compact               # Free up context window
/yolo                  # Auto-approve all tools
The second way is programmatic mode — pass a prompt directly with `-p`. This is perfect for scripts, CI/CD pipelines, and batch operations. The key commands are worth knowing: `--continue` resumes your last session exactly where you left off. `/compact` frees up context window space in long sessions. And `/yolo` — which toggles auto-approval of all tools — is fun but use with caution. Transition: Now let's talk about how tool approvals work.
