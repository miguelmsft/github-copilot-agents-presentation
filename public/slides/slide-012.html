Three Types of Tools

🔧
Built-in Tools
  • Read files
  • Write files
  • Run terminal
  • Search code
No setup needed
🔌
MCP Tools
  • Databases
  • APIs
  • GitHub
  • Playwright
  • Any MCP server
Configured via mcp.json
🧩
Extension Tools
  • VS Code extensions
  • Language Model API
VS Code only
There are three categories of tools. Built-in tools ship with Copilot — reading and writing files, running terminal commands, searching your codebase. These work everywhere. MCP tools come from external servers and let agents connect to databases, APIs, browsers, and more — we'll cover MCP in detail later. Extension tools are contributed by VS Code extensions and only work in the IDE. For most use cases, built-in and MCP tools are what you need.
