The Agentic Loop

Every Copilot agent follows the same pattern:

Understand → Act → Validate → Repeat

This loop is the engine that powers all agent types.

Regardless of where the agent runs — VS Code, terminal, or the cloud — they all follow this same loop. First, the agent gathers context by reading files and searching code. Then it acts — editing files, running commands. Then it validates — checking for errors, running tests. If something's wrong, it loops back. Let me show you this visually.
