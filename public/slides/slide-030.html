Third-Party Agents

Use Anthropic Claude or OpenAI Codex as alternative coding agents.

  • Same security protections as Copilot cloud agent
  • Bills through your existing Copilot subscription
  • Available locally (VS Code extension) or in the cloud
  • Claude: unique /agents, /hooks, /memory commands
  • Codex: can run interactively or unattended
GitHub doesn't lock you into one AI provider. You can use Anthropic's Claude or OpenAI's Codex as your coding agent. They run within the same framework and get the same security protections. Each provider brings unique strengths — Claude has its own sub-agent system and memory files, Codex has specialized coding capabilities. You pick the right tool for the job.
