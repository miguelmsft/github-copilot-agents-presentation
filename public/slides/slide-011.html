Tools: How Agents Interact with the World

Without tools, a language model can only generate text.

Tools are what let agents do things — read files, write code, run commands, and connect to external services.

This is a crucial concept. The language model — GPT, Claude, etc. — is the brain. But a brain without hands can't build anything. Tools are the hands. They let the agent interact with your development environment — read your code, make changes, run your tests. Without tools, you just get text. With tools, you get an autonomous developer.
