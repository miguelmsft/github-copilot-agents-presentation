What Are Hooks?

Hooks execute custom shell commands at strategic points in an agent's workflow.

Unlike instructions (which guide behavior), hooks guarantee that your code runs at specific moments.

Use cases:

  • 🛡️ Block dangerous commands before they execute
  • 📋 Create audit trails of every agent action
  • 🧹 Auto-format code after every edit
  • 🔔 Send Slack notifications on errors
Here's the key distinction. Custom instructions and skills are probabilistic — the AI usually follows them, but it's not guaranteed. Hooks are deterministic — your code always runs at the specified lifecycle point. If you need to block `rm -rf /` from ever executing, an instruction might work most of the time, but a hook guarantees it. Hooks are for enforcement, not guidance.
