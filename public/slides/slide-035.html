The architecture has three layers. The Host is your AI application — VS Code, Copilot CLI, Claude. The Host creates one Client per server connection. Each Client talks to exactly one Server. Servers are the providers — a database server, the GitHub server, a Playwright server for web browsing. Each server can expose tools, resources, and prompt templates. The agent discovers what's available and uses the right tools for the job.
