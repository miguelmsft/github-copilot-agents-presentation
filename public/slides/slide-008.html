What Is an Agent?

"An agent is an AI system that autonomously plans and executes coding tasks. You give it a high-level goal, and it breaks the goal down into steps, executes those steps with tools, and self-corrects when it hits errors."
— VS Code Documentation
This is the official definition from Microsoft. Three key words: plans, executes, and self-corrects. An agent doesn't just generate text — it takes actions in your development environment. It reads files, writes code, runs terminal commands, and checks its own work. It keeps going until the task is done or it needs your input.
