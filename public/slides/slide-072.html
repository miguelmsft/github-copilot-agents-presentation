Decision Tree: Which Mechanism to Use

What do you want to customize?
Standards
Consistent coding standards for ALL interactions → Custom Instructions .copilot-instructions.md
Templates
A reusable template for a specific task → Prompt Files .prompt.md
Workflows
A complex, multi-step workflow with scripts → Agent Skills SKILL.md
Personas
A specialized persona with tool restrictions → Custom Agents .agent.md
Enforcement
Deterministic enforcement at lifecycle points → Hooks .json
External
External API or database access → MCP Servers mcp.json
When in doubt, use this decision tree. Standards that apply everywhere? Custom instructions. A task you repeat with different inputs? Prompt files. A complex workflow with scripts? Skills. A specialized role? Custom agents. Enforcement guarantees? Hooks. External connectivity? MCP. In practice, you'll use several together — they're complementary layers, not competing alternatives.
