Configuring MCP in VS Code

Create .vscode/mcp.json:

{
  "servers": {
    "github": {
      "type": "http",
      "url": "https://api.githubcopilot.com/mcp"
    },
    "playwright": {
      "command": "npx",
      "args": ["-y", "@microsoft/mcp-server-playwright"]
    }
  }
}

Other ways to add servers: MCP Server Gallery, Command Palette, code --add-mcp, auto-discovery from other apps.

Here's a simple VS Code config with two servers — the GitHub remote server and a local Playwright server for browser automation. The GitHub server uses HTTP, while Playwright runs as a local subprocess via npx. VS Code also has an MCP Server Gallery where you can browse and install servers with one click, similar to the extension marketplace. You can also use input variables for secrets — never hardcode API keys.
