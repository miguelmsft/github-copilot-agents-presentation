Configuring MCP in Copilot CLI

Edit ~/.copilot/mcp-config.json:

{
  "mcpServers": {
    "playwright": {
      "type": "local",
      "command": "npx",
      "args": ["@playwright/mcp@latest"],
      "tools": ["*"]
    }
  }
}

Or use the interactive command:

/mcp add       → Fill in form → Ctrl+S to save
/mcp show      → See all configured servers
/mcp edit      → Edit a server config
The CLI format is slightly different — the root key is `mcpServers` instead of `servers`, and there's a `tools` field for filtering which tools are available. The interactive `/mcp add` command makes it easy — you get a form to fill in and save with Ctrl+S. The server is available immediately, no restart needed. Remember, the GitHub MCP server is already built in to the CLI.
