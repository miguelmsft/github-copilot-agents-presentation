🖥️ DEMO: Adding an MCP Server

🎮 LIVE DEMO

Demo Steps:

  1. Open VS Code → Extensions view → type @mcp
  2. Browse the MCP Server Gallery
  3. Install the Playwright MCP server
  4. Show .vscode/mcp.json created automatically
  5. In chat: "Navigate to github.com and take a screenshot"
  6. Watch the agent use the Playwright tool
[Live demo] Let me show you how easy it is to add an MCP server. We'll install Playwright from the gallery, which gives the agent browser automation capabilities. Then I'll ask it to browse to a website — watch how the agent calls the Playwright tools to navigate and take screenshots. This is how you extend what agents can do beyond just reading and writing files.
