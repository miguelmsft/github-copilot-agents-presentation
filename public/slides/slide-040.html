The GitHub MCP server is built into Copilot CLI — no setup needed.

MCP is supported across all three agent surfaces, each with its own config file. VS Code uses `.vscode/mcp.json`, the CLI uses `~/.copilot/mcp-config.json`, and the cloud agent is configured in repo settings on GitHub.com. The GitHub MCP server comes pre-installed in the CLI, giving you immediate access to issues, PRs, and repository data. The config formats are slightly different between surfaces, but the concept is the same.
