Cloud Agent — What It Is

An autonomous, cloud-hosted agent that runs on GitHub's infrastructure.

  • Works in the background on GitHub Actions runners
  • Creates draft pull requests for team review
  • Includes built-in security scanning (CodeQL, secret scanning)
  • Previously called "Copilot Coding Agent" — rebranded April 2026
The cloud agent is the most autonomous option. It runs entirely on GitHub's servers — you assign it a task and walk away. It researches your codebase, makes changes, runs your tests, scans for security issues, and opens a draft PR. Everything happens in commits and session logs, so your whole team can see what it did and provide feedback. This is like having a developer working overnight and leaving a PR for you to review in the morning.
