# GitHub Actions Failure Debugging

To debug failing workflows in a pull request:

1. Use `list_workflow_runs` to look up recent runs
2. Use `summarize_job_log_failures` to get an AI summary
3. If needed, use `get_job_logs` for full detailed logs
4. Try to reproduce the failure locally
5. Fix the failing build and verify before committing
