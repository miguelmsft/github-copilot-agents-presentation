Three Modes: Agent, Plan, Ask

GitHub Copilot Chat gives you three distinct modes:

  • 🤖 Agent Mode — Autonomous task execution (the star of the show)
  • 📋 Plan Mode — Create detailed implementation plans before coding
  • ❓ Ask Mode — Answer questions without changing any files
When you open Copilot Chat, you can choose between these three modes. Agent mode is what we've been discussing — full autonomous execution. Plan mode is like hiring a consultant to create a plan before any work starts — great for complex tasks where you want to review the approach first. Ask mode is traditional Q&A — no file changes, just answers. You can even hand off from Plan to Agent — approve the plan, then let the agent implement it.
